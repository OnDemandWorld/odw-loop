# Loop — Built

Loop is the orchestration layer that wires the suite's agents together into automated, multi-step workflows. Rather than competing head-on with general-purpose automation tools like n8n, Loop's distinct role is to be the conductor of your sovereign agent suite — connecting Vault, Desk, Recap, and the rest into pipelines that run on your own infrastructure.

## Differentiator
Think "your meeting notes become action items that update your knowledge base and trigger a follow-up," all without data leaving your control. Visual, low/no-code workflow building so non-engineers can compose agent automations, with the option to drop into code for advanced logic.

## Target
Teams that want sovereign automation across the ODW suite.

## Monetization
Free core lets a team build and run workflows on a single instance; paid layer adds cross-module orchestration at scale, unified control plane, high-availability deployment, and premium connectors.

## Positioning
Not "another Zapier" — the only orchestrator built specifically to coordinate ODW's own sovereign agents end to end.
